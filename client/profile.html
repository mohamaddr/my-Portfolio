Self Driving Car & Image Recognition

Wall-E automated parking system - University project where I got to work with C/python,Arduino, raspberry pi , openCV. Weve created a fully functional car code with C++ that does auto parking and manual controls. With this in mind, we have managed quite well with learning the C++ language. Same goes with arduino, weve got the code to work and function together with the arduino board.

System components:

The car is one of our main actor(s), since we are developing the software for it. The autoPark() is the main operation that we developed. The manualDrive() is one of our requirements for the car. The exitSpot() is to drive out of the parking spot. Application has the main operation of allowing the user to manual drive the car, via bluetooth which is specified as an attribute of the application. liveFeed() is to stream what the camera captures and show it on the application display. User is the person that will handle and drive the car through the application. Obstacles is what we consider to be other parked cars. The height/width/length is to define the minimum requirement of the obstacles. Camera with the operation to detect objects in front of it. And the liveStream() to send the footage to the application.
